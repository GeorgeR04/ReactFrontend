# GEM Gaming Esport Major — Frontend

Frontend web for the **GEM Gaming Esport Major** portfolio project, built with **React**, **Vite**, and **Tailwind CSS**.
It provides the user interface for tournaments (Major / Masters / Pro Circuit), teams, and authentication pages.
